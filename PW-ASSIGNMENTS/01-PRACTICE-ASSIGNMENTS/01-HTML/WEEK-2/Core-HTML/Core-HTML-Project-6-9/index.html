Question 6:

  1. HTML tag: 
    The HTML tag is the root tag within which the whole HTML document is written.
  2. Head tag: 
    The Head tag is the tag inside which all the meta information about the webpage is written.
The external links are added within this tag.
  3. Body tag: 
    The Body tag is the tag where the actual content of the website is written.
This is what gets displayed on the webpage by the browser.
  4. Heading tag: 
    The Heading tag is used to give the webpage a heading.
It ranges from h1 to h6, where h1 is the most important heading and h6 is the least.
  5. Paragraph tag: 
    The Paragraph tag is used to structure any written text into a paragraph.
  6. Anchor tag: 
    The Anchor tag is used to add a hyperlink in the website.
It has an attribute called href that specifies the url.
  7. Division tag: 
    The Div tag is used to group multiple elements to style them all at once.
  8. Image tag: 
    The Img tag is used to add an image to the webpage.
