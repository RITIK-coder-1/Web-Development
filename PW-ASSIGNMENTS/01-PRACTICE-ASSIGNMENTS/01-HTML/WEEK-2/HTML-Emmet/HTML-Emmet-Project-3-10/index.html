Question 3:

TAGS:
    HTML tags are used to create HTML documents and render their properties.
    They structure the webpage and help with the semantics.
    There are two types of tags basically, one that hold some content and that do not.
    The content holding tags have an opening part and a closing part,
    The another one is self closing.
    The HTML tag is the root tag within which everything is written.
    The head tag is the tag that contains the title and other meta information.
    The body tag is the tag where the actual content is displayed.
ELEMENTS:
    An HTML element is an HTML tag with some content inside it.
    The HTML element is everything from the start tag to the end tag:
    < tagname >Content goes here...< / tagname >. 
    
ATTRIBUTE:
    Attributes provide an additional information about the elements.
    Attributes are always specified in the start tag
    Their syntax goes as follows: The Attribute name="value".
    Some common attributes are the style attribute, used for styling, the anchor attribute, used to add a hyperlink.

Question 4

The heading tag: It is used to give a heading inside the webpage.
It has 6 types - h1 to h6. The h1 is the most important one and appears the boldest-largest by default.
The h6 is the least important one and appears the thinnest-shortest by default.

The paragraph tag: It is used to display a paragraph.
By default, there is no line break in this tag. We would have to use a break tag or a pre tag.

The div tag: It is used to divide other HTML elements into a section.

Question 7:

Nesting operators using emmet are used to position abbreviation elements inside generated tree: whether it should be placed inside or near the context element.
    
  • CHILD: You can use > operator to nest elements inside each other: h2>p>b 
    This will produce: (< h2 >< p >< b >< /b >< /p >< /h2 >)
    • 
    
  • SIBLING: Use + operator to place elements near each other, on the same level: div+p+bq
    This will output :
    < div >< /div >
    < p >< /p >
    < blockquote >< /blockquote >
    • 
    
  • Climb Operator: With ^ operator, you can climb one level up the tree and change context where following elements should appear: div+div>p>span+em^bq
    ...outputs to
        
        < div >< /div >
        < div >
            < p >< span >< /span >< em >< /em >< /p >
            < blockquote >< /blockquote >
        < /div >
    • 
    
  • Multiplication Operator: With * operator you can define how many times element should be outputted:
        ul>li*2 : 
        < ul >
            < li >< /li >
            < li >< /li >
        < /ul >
    
    • 

    ...ETC.

Question 9:

Self closing tags are the void elements that do not hold any content and generally, have no ending tag. They can have attributes. The following is a complete list of the void elements in HTML : area , base , br , col , command , embed , hr , img , input , keygen , link , meta , param , source , track , wbr.
    Meta tags are special type of void tags that provide the meta information about the web page and are generally wrriten inside the head tag.
