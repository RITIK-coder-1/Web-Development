HTML HTML5
  • HTML does not have attributes like data-, offline, onabort, onblur etc.
    •
  • HTML5 has additional attributes like data-, offline, onabort, onblur etc.
    •
  • In HTML, you can only find page layouts tags like div, span, etc.
    •
  • In HTML5, there are many more tag options available like header, footer, article, section, etc.
    •
  • HTML used the concept of cookies to store temporary data.
    •
  • HTML5 provides more flexible storage options. It is capable of using a user-side database like SQL. It also allows the user to view previously visited page data in offline mode.
